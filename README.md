# Fideli

Plataforma de fidelização digital para pequenos negócios.

## Funcionalidades atuais

- Login da empresa
- Cadastro de clientes
- Cadastro de serviços
- Registro de atendimentos
- Pontuação por cliente
- Área do cliente
- Visualização de pontos
